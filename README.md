# Cupon valido

**Nivel:** Fácil

## Descripción
Validar un código de cupón con base en su fecha de expiración y uso previo.

## Objetivo
Implementa una solución en Java que cumpla con la lógica descrita. Usa la plantilla `Main.java` para comenzar.

## Cómo empezar
1. Clona este repositorio.
2. Dirígete a la carpeta `Cupon_valido`.
3. Abre `Main.java` y escribe tu solución.
4. ¡Comparte tu solución con la comunidad!
